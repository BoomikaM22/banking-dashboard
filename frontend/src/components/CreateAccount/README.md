A sophisticated, 5-step digital onboarding system built with React. This component handles the end-to-end process of opening a bank account, including personal details, financial compliance (KYC/PEP), security credentials, document uploads, and initial deposit processing.Multi-Step Progress Tracking: Visual indicator showing "Personal," "Financial," "Security," "Documents," and "Payment" stages.

Real-time Validation: * Age verification (18+ only).

Password complexity (Uppercase, lowercase, numbers, and symbols).

Regex-based checks for Email, Phone, and Tax IDs.

Dynamic UI: Conditional rendering for employment details and payment methods (UPI vs. Card vs. NetBanking).

Document Handling: Support for file uploads for KYC verification.

Account Generation: Simulated backend processing that generates unique account numbers and assigns branch-specific IFSC codes upon completion
